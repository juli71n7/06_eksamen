Inspiration til portfoliesite

Moodboard

Inspirationen for mit portfolie site udkommer af desktop research på moderne og kreative websites. Endvidere af varme efterårsfarver med både komplimenter farver, monokrome farver og accentfarver. Mit portfoliesite er bygget op om den minimalistiske stilart, men med enkelte brud på stilartens regler for et unikt udtryk, som afspejler mig og mit arbejde. På sitet ses elementer med stærke udtryk, som er inspireret af tegnestilen one line drawings, som udtrykker det abstrakte i det ellers strukturerede og giver en form for retrostil. For et konsistent og harmonisk layout er alle wireframes bygget op omkring symmetri, således, at sitet virker overskueligt

Styletile

På styletilet ser du hjemmetegnet ikoner/logoer for at give portfoliesitet et legende, levende og kunstnerisk udtryk, som er med til at gøre websitet visuelt interesant at navigere rundt på. Der er gjort brug af gestaltlove så som: loven om nærhed og loven om lighed, loven om lukkethed for at skabe mening i mønstre og former, samt at give en sammenhæng på websitet

Der er gjort brug af 3 fonte til copy og micro copy. Alle fonte er sans serif, for at bibeholde elementer af den minimalistiske stilart og for at give portfoliesitet et moderne og nemt læseligt udseende.

Wireframes

Mit logo er designet ud fra one line/single line stilen, som er inspireret af picasso, der i det tidlige 20 århundrede tegnede en hest i en ren linje, for at udtrykke sin kunst på en minimalistisk måde. Denne tegnestil er brugt for at understøtte min delvist minimalistiske webdesign-stilart, og for at visualisere min personlige favorit tegnestil. Logoet skal ligne mig og repræsenterer mine mest genkendelige karaktertræk og mit kunsteriske sind.

Javascript

Jeg har i denne opgave arbejdet med Java script i form af en burger menu, samt enkelte animationer på mine “færdigheds-barer” på mit CV. Optimalt var mit ønske, at mine skillbar-animationer først skulle begynde, når de var visuelt indenfor skærmen. Eftersom Java script ikke er min stærke side, valgte jeg at holde mig til det jeg kunne og kan forklare.

Jeg har arbejdet med id attributter, samt container og sprite, på mine skillbarer og tilføjet classes (.classList.add(“tid”);) igennem java scriptet. Hvorimod jeg på min burgermenu har tilføjet classes i Html og referert til dem i java scriptet:
const burger = document.querySelector(“.burger”);
const navMenu = document.querySelector(“.nav-menu”);

HTML og CSS

Til opsætningen af mit portfoliesite har jeg brugt HTML, som er den grundlæggende konstruktion og struktur af et website. Derudover har jeg brugt CSS, som gør det muligt at ‘style’ websitets udtryk, altså ændre farve, tekststørrelse, afstande osv.

Portfoliesitet er opbygget efter den grundlæggende struktur med: head, body, header, main og footer og der er gjort brug af HTML tags så som div, nav, ul, img m.m. endvidere er der brugt CSS-variabler, id og classes, for en systematisk overskuelighed i koden. Det sværeste for mig er brugen af grid, herunder rows og columns, som jeg fremover gerne vil blive bedre til.
